Backend — POST /api/wallet/sign                                                                                                                                                                                                                                                                                                                                                                                                        
  Archivo nuevo: Frontend/src/pages/api/wallet/sign.ts                                                                                                                                                                
  
  Sigue el mismo patrón que send.ts (que ya funciona con Shamir):                                                                                                                                                                                                                                                                                                                                                                           
  1. Verificar auth (JWT) → user.id
  2. Buscar wallet en wallets → wallet_id, stellar_address
  3. Buscar shares en recovery_signers (mismo query que send.ts línea 85-92)
  4. reconstructKey(signer as StoredShares) → secret key temporal
  5. Decodificar el XDR con TransactionBuilder.fromXDR()
  6. Validaciones de seguridad:
    - Source de la transacción === stellar_address del usuario
    - Bloquear accountMerge
    - Bloquear setOptions que modifique signers o masterWeight
  7. Firmar con la keypair
  8. Si submit: true → enviar a Stellar → devolver { signedXdr, txHash }
  9. Si submit: false → devolver { signedXdr }

  Imports que ya existen y reutilizo:
  - reconstructKey, StoredShares de services/shamir
  - getAuthUser, supabaseAdmin de services/supabase
